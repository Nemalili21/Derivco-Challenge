# MzansiBuilds - Derivco Code Skills Challenge

This is a public platform where developers can:

- Create and manage accounts
- Add projects with stages and support required
- View live feed of other developers' projects
- Update project progress with milestones
- Celebrate completed projects on a Celebration Wall

**Tech Stack:** HTML, CSS, Python (Flask), SQLite
